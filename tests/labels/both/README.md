# Combined Detection Labels

YOLO annotations for both fire and smoke. Class IDs: 0=fire, 1=smoke

Examples:
```
0 0.4 0.5 0.3 0.25
1 0.6 0.3 0.4 0.35
1 0.2 0.7 0.25 0.2
```
